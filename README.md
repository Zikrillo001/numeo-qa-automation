# Numeo QA Automation Challenge

## Overview
This repository contains a QA automation solution for a demo full-stack application with a Node.js backend and React frontend.

## Scope
- API testing
- End-to-end testing
- Flaky/async behavior handling
- CI-ready structure
- Observability support

## Tech Stack
- Playwright
- TypeScript
- GitHub Actions

## Project Structure
- `app/` — system under test
- `tests/api/` — API tests
- `tests/e2e/` — UI tests
- `src/api-clients/` — API abstraction layer
- `src/ui-pages/` — page objects
- `src/factories/` — test data generation
- `src/utils/` — helpers and config
